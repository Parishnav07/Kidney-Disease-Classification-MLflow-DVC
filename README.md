# Kidney-Disease-Classification-MLflow-DVC

# How to Run ?

### STEPS :-

Clone the repository

```bash
https://github.com/Parishnav07/Kidney-Disease-Classification-MLflow-DVC
```

### STEP 1 : - Create a conda virtual environment after opening the repo

```bash
conda create -n cnncls python=3.8 -y
```

```bash
python3 -m venv cnncls
```

Activate the environment
```bash
source cnncls/Scripts/activate
```

### STEP 2 :- Install dependencies using pip
```bash
pip install -r requirements.txt
```
